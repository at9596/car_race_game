# Car-Racer-Game
Make a racing game in Python using pygame!

# Clone this repo:
  ``` git clone git@github.com:at9596/car_race_game.git ```

 # Install the dependancies
   ``` pip install -r requirements.txt ```


# Move forward the car:
    ```  key: 'w' -> move forward
            's' -> move backward
            'a' -> move left
            'd'  -> move right
